# EVOLUÇÃO - AGENDA MOBILE, WHATSAPP E STATUS

## Decisões aplicadas
- A operação foi simplificada para uso majoritário no celular.
- WhatsApp continua sendo o canal principal de execução.
- Contrato aceito não mostra mais preparo de aceite.
- Reserva aguardando pagamento mostra itens e informar pagamento.
- Agenda agora mostra uma reserva por vez, ordenada pela hora do evento.
- Previsão operacional passou a ser automaticamente a hora informada até +1h.
- Campo de localização fica apenas para colar o link enviado pelo motorista no WhatsApp.
- Correção defensiva para valores inflados em base de teste por máscara monetária antiga.

## Fluxo reforçado
Contrato → Reserva → Agenda → Informar pagamento → Financeiro
